# Google Calendar Bulk Delete in French

Bulk delete Google Calendar events by search term. Deletes ALL events containing the entered text.

![Demo](demo.gif)

## Usage

1. Open Google Calendar
2. Run script in browser console
3. Enter search text (or many words separed by commas) when prompted
4. Confirm deletion

## Features

- Processes to desired months or weeks
- Handles recurring events
- Progress tracking in console
- Search using various keywords

**Warning:** Permanent deletion. Cannot be undone.
