# Google Calendar Bulk Delete / Suppression en masse d'événements Google Calendar

Bulk delete Google Calendar events by search term.

---

## Version française

Supprime en masse des événements Google Calendar à partir de mots-clés.

![Demo](demo.gif)

### Utilisation

1. Ouvrir Google Calendar
2. Coller le script dans la console du navigateur (F12 ou ctrl + maj + i). Attention, par défault il y aura une erreur pour des raisons de sécurité (coller du code inconnu), il faut alors dans ce cas écrire "alow pasting". 
3. Entrer un ou plusieurs mots (séparés par des virgules). Exemple : La chaine "et je, alors" supprimera tout ce qui contient "et je" et "alors" dans le nom.
4. Confirmer la suppression

### Fonctionnalités

- Parcourt plusieurs semaines ou mois
- Gère les événements récurrents
- Affiche la progression dans la console
- Recherche avec plusieurs mots-clés

**Attention :** suppression définitive, impossible à annuler. La limite de l'outil est que si vous voulez supprimer "cours", cela supprimera aussi bien "cours de maths" que "secours" !!

## English version

Bulk delete Google Calendar events using search terms.

![Demo](demo.gif)

### Usage

1. Open Google Calendar
2. Run the script in your browser console
3. Enter one or more keywords (comma separated)
4. Confirm deletion

### Features

- Processes multiple weeks or months
- Handles recurring events
- Progress logs in console
- Supports multiple search terms

**Warning:** permanent deletion. Cannot be undone.
